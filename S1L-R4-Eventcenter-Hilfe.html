1 · Das große BildThe big pictureGenel resimОбщая картина

Du pflegst nur eine Liste (die Event-Auswahl). Daraus entstehen automatisch drei Ausgaben — je nachdem, welche Schalter an sind.You maintain only one list (the event selection). From it three outputs are generated automatically — depending on which switches are on.Sadece tek bir liste tutarsın (etkinlik seçimi). Hangi anahtarların açık olduğuna göre üç çıktı otomatik oluşur.Ты ведёшь только один список (выбор событий). Из него автоматически создаются три результата — в зависимости от включённых переключателей.

Kurz: „Aktiv" entscheidet, ob ein Event überhaupt läuft. „Mitteilen" entscheidet, ob es in den Chat geht. Termin-Events mit Zeit erscheinen zusätzlich als Live-Ansage.In short: "Active" decides whether an event runs at all. "Announce" decides whether it goes to chat. Timed events with a time also appear as a live call.Kısaca: "Aktif" etkinliğin çalışıp çalışmadığını belirler. "Duyur" sohbete gidip gitmediğini belirler. Zamanlı etkinlikler ayrıca canlı duyuru olarak görünür.Коротко: «Активно» решает, идёт ли событие вообще. «Сообщить» решает, попадёт ли оно в чат. События со временем дополнительно показываются как live-объявление.

5 · Tagesbriefing (Brett)Daily briefing (board)Günlük brifing (pano)Дневной брифинг (доска)

Die eine Nachricht fürs Allianz-Brett. Sie baut sich aus allen Blöcken mit „Mitteilen" — Reihenfolge ▲▼ = Priorität. Die Länge stellst du pro Event unten in der Auswahl ein (kurz/lang) — so kann z. B. Ghulöwe kurz und der FOKUS lang sein. Es gibt keinen globalen Kurz/Voll-Schalter mehr.The one message for the alliance board. It is built from all "Announce" blocks — order ▲▼ = priority. You set the length per event below in the selection (short/long) — e.g. Ghoulion short, the FOCUS long. There is no global short/full switch anymore.Pano için tek mesaj. "Duyur" bloklarından oluşur — sıra ▲▼ = öncelik. Uzunluğu etkinlik başına aşağıda ayarlarsın (kısa/uzun) — örn. Ghoulion kısa, ODAK uzun. Artık global kısa/tam anahtarı yok.Одно сообщение для доски. Собирается из блоков «Сообщить» — порядок ▲▼ = приоритет. Длину задаёшь по событию ниже (кратко/полно) — напр. Ghoulion кратко, ФОКУС полно. Глобального переключателя больше нет.

  • kurz / short / kısa / кратко: Event-Name + Kernaussage in einer Zeile (= Kurzlabel der Strategie) + Zeit (voller Länder-Block), solange das Event nicht vorbei ist.event name + key point in one line (= strategy's short label) + time (full country block), as long as it isn't over.etkinlik adı + tek satır özet (= stratejinin kısa etiketi) + zaman (tam ülke bloğu), bitene kadar.название + суть в одну строку (= краткая метка стратегии) + время (полный блок), пока не завершено.
  • lang / long / uzun / полно: zusätzlich die volle Strategie. Meist nur fürs wichtigste Event (FOKUS) — sonst wird die Nachricht zu lang.additionally the full strategy. Usually only for the most important event (FOCUS) — otherwise the message gets too long.ek olarak tam strateji. Genelde sadece en önemli etkinlik (ODAK) için.дополнительно полная стратегия. Обычно только для главного (ФОКУС).

  2. Pro Event eine Strategie wählen (★ = empfohlene Prio).Pick a strategy per event (★ = recommended priority).Etkinlik başına strateji seç (★ = önerilen).Выбери стратегию по событию (★ = рекомендуется).
  3. Was aufs Brett soll: Mitteilen an. GW und NAP nach Bedarf dazuschalten/ausfüllen.What goes to the board: turn on Announce. Add/fill GW and NAP as needed.Panoya gidecekler: Duyur aç. GW ve NAP'ı gerektikçe ekle/doldur.Что на доску: включи Сообщить. Добавь GW и NAP при необходимости.
  4. Mit ▲▼ die Reihenfolge = Priorität setzen (oberstes = FOKUS).Set order = priority with ▲▼ (top = FOCUS).▲▼ ile sırayı = önceliği ayarla (en üst = ODAK).Задай порядок = приоритет через ▲▼ (верх = ФОКУС).

8 · Wichtige RegelnImportant rulesÖnemli kurallarВажные правила

  • Nur Entwürfe. Das Eventcenter sendet nichts. Du kopierst und fügst selbst ein.Drafts only. The event center sends nothing. You copy and paste yourself.Sadece taslak. Merkez bir şey göndermez. Sen kopyalayıp yapıştırırsın.Только черновики. Центр ничего не отправляет. Копируешь и вставляешь сам.
