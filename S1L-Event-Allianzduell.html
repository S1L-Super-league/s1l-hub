📌 Das Wichtigste

🎯 Ziel: Mehr Tagespunkte als der Gegner → Tag gewinnen. 13 Punkte gesamt, 7 zum Sieg. FIX

🏆 Belohnung: Tagesaufgaben → persönliche Meilenstein-Truhen (immer, sieg-unabhängig) + Allianz-Sieg = reiche Truhen. FIX

⚙️ Mechanik: Jeder Tag ein Thema mit Gewicht. Tag 6 „Gegner besiegen" zählt 4 → da fällt die Entscheidung. FIX

⛔ NICHT: seltene Ressourcen an niedrig gewichteten Tagen verbrennen — für den passenden Tag / das Golden Window aufheben. FIX

🧠 Bevorzugte Strategie — warum & wie

Der Sieg-Einsatz gehört auf hoch gewichtete, gewinnbare Tage — vor allem Tag 6 (Gewicht 4, Gratis-Versetzung). Persönliche Belohnungen kommen aus eigenen Punkten und sind sieg-unabhängig → Tagesaufgaben lohnen immer. Eine Niederlage kostet nur eine kleinere Truhe, keine Ressourcen-Strafe.

💾 Spar-Plan — was für welchen Tag aufheben
  • Ausrüstungsfetzen → Rüstungswettlauf-Ausrüstungsrunde / Turbo-Schildkröte (doppelt)
  • Bau-Speedups → Tag 2 · Forschungs-Speedups + Ungeheuer-Zellen → Tag 3
  • Heldenfragmente + Rekrut-Marken → Tag 4 · Truppen-Training → Tag 5
  • Friedenskuppeln → Tag 6
