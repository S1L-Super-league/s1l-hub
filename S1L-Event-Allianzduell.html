⚔️ Allianzduell

6-Tage-Duell Allianz gegen Allianz, 1 Thema pro Tag. Tag 6 entscheidet. FIX

📌 Das Wichtigste

🎯 Ziel: Mehr Tagespunkte als der Gegner → Tag gewinnen. 13 Punkte gesamt, 7 zum Sieg. FIX

🏆 Belohnung: Tagesaufgaben → persönliche Meilenstein-Truhen (immer, sieg-unabhängig) + Allianz-Sieg = reiche Truhen. FIX

⚙️ Mechanik: Jeder Tag ein Thema mit Gewicht. Tag 6 „Gegner besiegen" zählt 4 → da fällt die Entscheidung. FIX

✅ Was tun / ⛔ NICHT tun

Tun: jeden Tag die Tagesaufgaben abräumen (garantierte Meilenstein-Truhen). FIX

⛔ NICHT: seltene Ressourcen an niedrig gewichteten Tagen verbrennen — für den passenden Tag / das Golden Window aufheben. FIX

💡 Punkte-Boost durch Forschung

Wichtig: Die Punktzahlen oben sind Basiswerte. Im Spiel siehst du deine eigenen, geboosteten Zahlen (grün) — jeder hat andere, je nach Forschung. FIX

Mehr Punkte holen: Im Labor die Forschungen „Kampfexperte" (alle Duell-Punkte) und die „Fokus"-Techs (Beschleuniger, Radar, Helden …) leveln — je Stufe +5 %, bis +50 %. Speedups bekommen „Alle Punkte" + „Beschleuniger" doppelt.

Deine genauen Boni: Forschung öffnen → beim jeweiligen Punkt auf das ? tippen → dort steht der exakte %-Wert. Käufe (Diamanten) werden nicht geboostet.

🎁 Belohnungen

3 Ebenen: persönliche Meilenstein-Truhen (Punkte sammeln) · tägliche Rang-Belohnung (dein Tagesrang) · Allianz-Belohnung (Teilnahme/Sieg/Niederlage). FIX

🤝 Allianz-Belohnungen
  • Teilnahme (täglich): 10× 10K-EXP + Ressourcen — ab 500.000 Tagespunkten.

🎖️ Kampfmedaille (VS-Medaille)

Was: die rote VS-Medaille — Duell-Währung. Bekommst du aus fast jeder Duell-Belohnung (Truhen 50–400, Tagesrang 100–300, Allianz-Sieg 800 / Niederlage 400).

Wofür: An Tag 3 gibt „1 Kampfmedaille ausgeben" +2.500 Punkte (Basis) → vorher sammeln, an Tag 3 ausgeben.

Wo ausgeben: sehr wahrscheinlich im Labor — damit levelst du die VS-/„Fokus"-Forschungen, die deine Duell-Punkte boosten. final im Client bestätigen

🧠 Bevorzugte Strategie — warum & wie

Der Sieg-Einsatz gehört auf hoch gewichtete, gewinnbare Tage — vor allem Tag 6 (Gewicht 4, Gratis-Versetzung). Persönliche Belohnungen kommen aus eigenen Punkten und sind sieg-unabhängig → Tagesaufgaben lohnen immer. Eine Niederlage kostet nur eine kleinere Truhe, keine Ressourcen-Strafe.

💾 Spar-Plan — was für welchen Tag aufheben
  • Ausrüstungsfetzen → Power Play (Gear-Enhancement-Runde) / Turbo-Schildkröte (doppelt)
  • Bau-Speedups → Tag 2 · Forschungs-Speedups + Ungeheuer-Zellen → Tag 3
  • Heldenfragmente + Rekrut-Marken → Tag 4 · Truppen-Training → Tag 5
  • Friedenskuppeln → Tag 6
